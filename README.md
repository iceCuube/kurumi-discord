# kurumi-discord

run with
`python mainKurumi.py [discordkey] [osukey]`

# all commands

## osu
```
u&osurecent
u&osubest
u&osuplayer
```

## tic tac toe
```
u&tictactoe
u&accept
u&decline
```

## fun
```
u&say
u&randomkurumi
u&eightball
u&coinflip
u&rps
u&hug
u&secks
```

## music
```
u&connect | u&con
u&disconnect | u&dcon
u&play | u&p
u&skip | u&s
u&stop
```

you can thank her by saying "thanks kurumi"
